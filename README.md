# 🥛 MilkMaster

MilkMaster is a modern Dairy Management System built with Flask to replace the traditional notebook-based milk distribution system.

## Features

- 👥 Customer Management
- 🥛 Daily Milk Entry
- 💰 Payment Tracking
- 📊 Reports
- 🐄 Cattle Management
- 🤰 Pregnancy Tracking
- 💸 Expense Management
- 📄 PDF Statement Generation
- 📱 Responsive Mobile Interface
- 🌐 Progressive Web App (PWA)
- 💾 Backup & Restore

## Technologies Used

- Python
- Flask
- SQLAlchemy
- SQLite
- Bootstrap 5
- HTML
- CSS
- JavaScript

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/MilkMaster.git
cd MilkMaster

python -m venv .venv

# Windows
.venv\Scripts\activate

pip install -r requirements.txt

python app.py
```

The application will start locally and can be opened in your browser.
